# devconfig

Mac 개발 환경 설정 파일 모음

## Tools

| 도구 | 설명 | 설정 경로 |
|------|------|-----------|
| [Hammerspoon](hammerspoon/) | 키보드 단축키로 앱 실행 | `~/.hammerspoon/` |
| [Pet](pet/) | CLI 명령어 스니펫 관리 | `~/.config/pet/snippet.toml` |
| Fig | 터미널 자동완성 | `~/.fig/settings.json` |
| Rectangle | 윈도우 크기/위치 조절 | - |

## Usage

```bash
# 최초 설치
./install.sh

# 레포 수정 후 로컬에 반영
./sync.sh
```
