# Rectangle

키보드 단축키로 윈도우 크기와 위치를 조절하는 도구.

https://rectangleapp.com

## 설치

```bash
brew install rectangle
```

## 기본 설정

첫 실행 시 `시스템 환경 설정 - 보안 및 개인정보 보호 - 손쉬운 사용`에서 권한을 허용해야 한다.

## 단축키

| 단축키 | 동작 |
|--------|------|
| `⌘ + ⌥ + ←` | 왼쪽 절반 |
| `⌘ + ⌥ + →` | 오른쪽 절반 |
| `⌘ + ⌥ + ↑` | 최대화 |
| `⌘ + ⌥ + ↓` | 원래 크기 복원 |
